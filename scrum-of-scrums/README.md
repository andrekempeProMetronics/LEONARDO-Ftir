# Project Standup — LEONARDO-Ftir

Project-internal sync meeting. The cross-project Scrum of Scrums lives in
`PROM-Portfolio/scrum-of-scrums/`.

## Cadence

_TBD — typically weekly._

## Attendees

LEONARDO-Ftir project team.

## Standard agenda

1. **Status round-robin** — what shipped, what's planned next.
2. **Blockers** — anything stalling work (technical, organizational, dependencies on other sub-projects).
3. **Cross-team dependencies** — items to escalate to the portfolio scrum-of-scrums.
4. **Action items** — concrete next steps with owner and due date.

## Notes convention

- One file per meeting under `notes/`, named `YYYY-MM-DD.md`.
- Use the agenda above as the section structure.
- Action items at the bottom in a checklist; carry unfinished items forward.
