## Download

The Cowork sandbox cannot reach the Leonardo DRS CDN, so the PDFs are fetched by a script you run locally:

```bash
cd "docs/00_Project_Input/publications"
bash download_spero_publications.sh
```

Files are named `NN_YEAR_short-title.pdf`. Re-running re-fetches; delete a file to force a fresh copy.
